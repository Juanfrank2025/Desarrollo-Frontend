## 📝 Descripción del proyecto

**Ruta de Paz** es un sitio web informativo y promocional para un alojamiento turístico ubicado en el departamento de Morazán, El Salvador. El sitio permite a los visitantes conocer los servicios de hospedaje (cabañas, camping, hoteles boutique), explorar sitios turísticos cercanos y realizar reservaciones mediante un formulario validado con envío real de correos vía Formspree.

El diseño es **fully responsive**, adaptándose a dispositivos móviles, tablets y escritorio.

---

## 🛠️ Tecnologías utilizadas

- **HTML5** — Estructura semántica con 6 secciones obligatorias
- **CSS3** — Diseño responsive con Grid, Flexbox, variables CSS y animaciones
- **JavaScript** — Validación de formularios, menú hamburguesa, scroll spy, modal de éxito
- **Formspree** — Backend para envío de correos de reservación sin servidor propio
- **Font Awesome** — Iconografía de redes sociales y servicios
- **Google Fonts** — Tipografías Montserrat y Open Sans
